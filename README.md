# Pathfinder SVG

A simple svg editor & generator

**This is Demo Version**

## Demo

execute

```
$ git clone https://github.com/RudyPark3091/pathfinder-svg.git
$ cd pathfinder-svg
$ npm install
$ npm run build
$ npm install serve
$ serve -s build/public
```

and go to http://localhost:5000

also checkout desktop application version built with electron

```
$ npm start
```
